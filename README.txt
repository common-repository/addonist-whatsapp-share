Add WhatsApp button share into posts and WooCommerce products.

== Description ==

Install the plugin and the button will display automatically into every post. 
Also you can use shortcode [addonist_whatsapp].

PRO VERSION:

* Share WooCommerce products
* Get statistics about every share

For more details: https://www.addonist.com/shop/whatsapp-share/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/addonist-whatsapp-share` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Locate Addonist -> Whatsapp Share

* Use the [addonist_whatsapp] shortcode if you need

== Frequently Asked Questions ==
